# Solana Contracts

Solana smart contracts (programs) for Crossify.io using Anchor framework.

## Setup

1. Install Anchor: https://www.anchor-lang.com/docs/installation

2. Install dependencies:
```bash
anchor build
```

3. Deploy to devnet:
```bash
anchor deploy --provider.cluster devnet
```

## Programs

- **token_factory**: Creates SPL tokens
- **bonding_curve**: Bonding curve for token sales
- **migration**: Migration to Raydium DEX

For full Solana implementation, use Anchor framework with proper program structure.
